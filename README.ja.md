# AE — フォルダをそのままプロジェクトパネルに取り込む

**v1.0** · [English](README.md) · [한국어](README.ko.md)

ディスク上のフォルダを選ぶと、**プロジェクトパネル**に同じ階層ができ、その中に読み込みます。

**macOSのみ検証**（AE 2026）。Windowsは未確認。

**インストール:** `marie-import-folder-structure.jsx` を例として  
`/Applications/Adobe After Effects 2026/Scripts/` へ  
環境設定→スクリプトで **ファイルの書き込み・ネットワークを許可**。

**実行:** **ファイル→スクリプト→marie-import-folder-structure**

イメージシーケンスはAE標準の取り込みと挙動が違う場合あり。日本語などのフォルダ名はパスから補正。
